# Backend Build Commands

## Windows (PowerShell)

- Run app: `mvnw.cmd spring-boot:run`
- Run all tests: `mvnw.cmd test`
- Run single test: `mvnw.cmd test -Dtest=TestClassName#methodName`
- Full build: `mvnw.cmd clean package`

## macOS/Linux

- Run app: `./mvnw spring-boot:run`
- Run all tests: `./mvnw test`
- Run single test: `./mvnw test -Dtest=TestClassName#methodName`
- Full build: `./mvnw clean package`
